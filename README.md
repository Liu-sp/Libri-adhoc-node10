# Libri-adhoc-nodes10
Libri-adhoc-nodes10 is a synchronized speech corpus, which collects the replayed Librispeech data from loudspeakers by ad-hoc microphone arrays of 10 synchronized distributed nodes in real world environment.
## Description of the dataset
The Libri-adhoc-nodes10 dataset is composed of the speech signals collected by the microphone array and use the "test-clean" part of the Librispeech as the sound source.  The "test-clean" part of the Librispeech contains about 5.4 hours of US English speech from 20 male speakers and 20 female speakers. This dataset is recorded in two scenarios, an office room and a conference room.
